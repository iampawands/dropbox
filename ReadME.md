# Dropbox Clone

This is a simple Dropbox-like application with separate frontend and backend components.

## Project Structure

- **dropbox-frontend/** – Frontend code written in **React** with **TypeScript**  
- **dropbox-backend/** – Backend code written in **Spring Boot**

---

## Prerequisites

### Backend
- Java 17+
- Maven 3.6+
- A running MySQL/PostgreSQL database (or compatible JDBC DB)

### Frontend
- Node.js (refer to `package.json` for the required version)
- npm (comes bundled with Node.js)
